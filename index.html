Flexbox Examples

Flexbox is a CSS layout tool that arranges items in a row or column. Changing just one property can completely change how things look. Explore the examples below to see each one in action.

display: flex + justify-content: center

Adding display: flex to a container turns it into a flexbox. Items automatically line up side by side. justify-content: center moves them to the middle of the row.

flex-direction: column

The default direction is row (left to right). Change it to column and items stack top to bottom instead. Great for things like sidebars or vertical menus.

justify-content: space-between

Pushes items as far apart as possible. The first item goes to the left edge, the last item goes to the right edge, and any remaining items are evenly distributed in between.

justify-content: space-around

Similar to space-between, but every item also gets equal space on both its left and right sides. This means the gap at the edges is half the gap between items.

align-items: center

justify-content controls alignment on the main axis (horizontal). align-items controls the cross axis (vertical). Setting it to center vertically centers all items — notice the short and tall boxes line up in the middle.

flex-wrap: wrap

By default, flexbox squishes all items into one row. Add flex-wrap: wrap and any items that don't fit automatically jump to the next line — like text wrapping in a paragraph.

flex-grow

flex-grow controls how much a flex item expands to fill leftover space. Box 1 and Box 3 have flex-grow: 1. Box 2 has flex-grow: 2 — so it grabs twice as much of the extra space.
